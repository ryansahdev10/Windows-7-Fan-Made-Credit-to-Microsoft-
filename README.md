## 🚀 Core Features

### 🖥️ Desktop & Interface
* **Aero Glass Theme:** Beautifully replicated translucent window borders and glass effects.
* **Fully Functional Taskbar:** Pin apps, minimize/maximize windows, and use the system tray clock.
* **Classic Start Menu:** Fully interactive Start Menu featuring user profile pictures, quick access links, and a functional search bar.
* **Right-Click Context Menu:** Refresh the desktop, sort icons, or change desktop wallpapers on the fly.

### 📁 File Explorer & Window Management
* **Interactive Windows:** Drag, maximize, minimize, resize, and close any application window.
* **File System Simulation:** Browse default folders like *Documents*, *Pictures*, *Music*, and *Computer*.
* **Desktop Icons:** Double-click shortcuts to launch apps, and drag icons to rearrange your desktop grid.

### 🎮 Built-in Applications & Tools
* **Internet Explorer:** A simulated web browser to surf a selection of preset or live websites.
* **Notepad:** A working text editor to type, edit, and clear notes.
* **Calculator:** Fully operational calculator with standard arithmetic functions.
* **Paint:** A basic canvas painting app to doodle just like the original MS Paint.
* **Minesweeper / Solitaire:** Nostalgic mini-games to kill time (if included in your build).
* **Media Player:** A simulated audio/video player pre-loaded with classic Windows sample media.

### ⚙️ System Customization
* **Control Panel:** Change the system theme, modify display settings, or swap out desktop backgrounds.
* **Desktop Gadgets:** Toggle classic gadgets like the analog Clock, Calendar, and CPU Meter on your desktop.
* **Sign-in & Shut Down Screens:** Authentic boot-up, user selection login screen, and logging-off sequences accompanied by the iconic original sound effects.
